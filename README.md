# Discord TTS Wizard 語音大師

Discord TTS Wizard 是一個專為實況主、Discord 使用者以及語音創作者打造的「文字轉語音 (TTS)」工具。它免安裝、開啟即用，讓您可以輕鬆地把打出來的文字變成高音質的 AI 語音，並同時發送到您的耳機以及語音頻道中 (須搭配 **VB-Audio Virtual Cable** 使用)！

## ✨ 主要特色
* **即時多通道播放**：可以一邊讓自己聽到語音，一邊把語音傳送給 Discord 的朋友聽。
* **零延遲管線化引擎**：就算輸入極長的文章，也會瞬間發聲，告別漫長的等待。
* **檔案匯出功能 (MP3 / WAV)**：不只即時播放，還能直接把產生的語音無損存成音檔，用於影片剪輯或 Podcast。
* **聰明記憶**：會自動記住您上次選擇的聲音、設定和設備，下次打開直接開講！

---

## 🚀 快速上手 (給非工程師的懶人包)

1. **取得軟體**：在資料夾內找到 `Discord_TTS_Wizard.exe`。
2. **開啟軟體**：對著它點兩下即可開啟，不需額外安裝任何程式！
3. **輸入文字**：在下方的大對話框內打字。
4. **一鍵發送**：點擊綠色的 `Play / Send` 按鈕（或直接按下鍵盤的 `Enter` 鍵），就能立刻聽到 AI 幫你把字唸出來！
   *(💡 提示：如果想在輸入框內換行，請按 `Shift` + `Enter`)*

## 📖 介面功能詳解

### 1. 聲音選擇 (Voice Model)
在這裡可以切換不同的 AI 說話者：
* **曉辰 (HsiaoChen)**：溫柔甜美的台灣女生聲音，適合日常聊天或說故事。
* **允哲 (YunJhe)**：清晰穩重的台灣男生聲音，適合正式報告或解說。

### 2. Live Playback (即時播放模式)
這是預設的操作模式，用來「即時」把聲音播送出去。最特別的是，你可以新增**多個輸出通道 (Output Devices)**。

* **如何操作**：
   * 點擊下拉選單，可以選擇你的喇叭或耳機。
   * 點擊右側的 `+ Add Channel`，會多出一排選單。
* **為什麼要有兩個通道？**
   如果你是 Discord 玩家或實況主：你可以把第一個通道設定為你的「實體耳機」，第二個通道設定為你的「Virtual Audio Cable (虛擬音源線)」。
   這樣一來，AI 講話的時候**你自己聽得到**，而 Discord 裡面的**朋友也聽得到**！
* *(程式有防呆機制，被選過的設備會自動在其他選單中隱藏，防止重複選擇！)*

### 3. Export to File (匯出檔案模式)
如果你今天不想直播，只是想把語音存下來剪片用，請點選這個分頁！
* **Format (格式)**：提供 `mp3` (檔案較小) 與 `wav` (無損高音質) 兩種選擇。
* 切換到此模式後，底下的按鈕會變成 **`Save to File`**。點擊後會彈出視窗讓你選擇要存在哪裡，非常直覺！

### 4. 便利小設定
* **Auto-Clear (自動清空)**：只要打勾，當你送出文字後，輸入框就會自動幫你清乾淨，方便你馬上打下一句話。這個狀態系統也會自動幫你記住喔！

## 🛠 如果你想讓 Discord 的朋友聽到...
**(需搭配虛擬音源軟體，例如 VB-Audio Virtual Cable)**
1. 安裝 VB-Audio Virtual Cable。
2. 開啟 Discord TTS Wizard，在 **Live Playback** 中新增兩個通道。
3. 一個通道選擇你的**耳機**，另一個通道選擇 **CABLE Input (VB-Audio Virtual Cable)**。
4. 打開 Discord 的語音設定，將**麥克風輸入 (Input Device)** 更改為 **CABLE Output**。
5. 搞定！你在軟體裡打字的語音，就會直接在語音頻道裡面播送啦！
